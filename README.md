# Tender CRM Desktop

Десктоп-приложение для управления тендерами и работы с AI-поиском.

## 🖥 Возможности

- **Управление тендерами** — создание, редактирование, фильтрация
- **Просмотр писем** — с предпросмотром и метаданными
- **AI Поиск (RAG)** — семантический поиск по документации с ответами LLM
- **Связывание** — автоматическое и ручное связывание писем с тендерами
- **Статистика** — сводка по системе в реальном времени

## 🛠 Технологии

- **Qt 6.10** — UI фреймворк
- **C++17** — основной язык
- **CMake** — система сборки
- **Qt Network** — HTTP-клиент для работы с API

## 📦 Сборка

### Требования

- Qt 6.10 или выше
- CMake 3.16+
- Компилятор: MSVC 2022 или MinGW

### Сборка в Qt Creator

1. Откройте `CMakeLists.txt` в Qt Creator
2. Выберите kit (Qt 6.10 + компилятор)
3. Build → Run (Ctrl+R)

### Сборка через CMake

```bash
mkdir build && cd build
cmake ..
cmake --build .
